# AI Agent Skills Directory

This file contains standard operating procedures (SOPs) and rules that AI Agents MUST follow when executing tasks in this repository.

## Skill: Writing Customer Emails
When you are asked to draft an email to a customer, you must follow this exact format:
1. Always start with: "Dear [Customer Name],"
2. If the topic is a refund or complaint, you must include the phrase: "We sincerely apologize for the inconvenience."
3. Always sign off the email exactly like this:
   "Warmly,"
   "The LangGraph Support Team"

## Skill: Writing Python Code
When you are asked to generate Python code for this project, you must adhere to these repository standards:
1. Always use Python type hints (e.g., `def calculate(a: int) -> int:`).
2. Every function MUST have a brief docstring explaining what it does.
3. You must use the `logging` module instead of `print()` statements for all console output.
